Projects

CharacterSheetBot

CharacterSheetBot is a Twitter bot that randomly generates a Dungeons and Dragons character, including name, stats, and backstory text. It then prints those stats onto an image of a D&D character sheet, and tweets it out. The bot is built in Python, using the Python Imaging Library and pytracery. All of the code can be found here on Github.
